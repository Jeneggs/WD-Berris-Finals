Purpose


This website has four main purposes. To serve as educational work, to improve my skills, to advertise myself, and express my skills in using HTML and CSS.

Educational Work


This website is made for educational purposes only. It serves as a final project to demonstrate all the skills I've learned in my Web Design course.

Skill Enhancement


As a final project, this website was made to be more challenging and required the application of all the skills I have learned so far and more. By creating this website, my skills in HTML and CSS have been enhanced in a way that I can create better websites.

Personal Advertising


This website also serves as a resume, advertising myself in a way. If there is anyone interested in using my skills, feel free to contact me which can be found it the about me page!

Skill Expression


This website serves as an art board that expresses my current skills in building websites. It shows what sets me apart from my peers.
